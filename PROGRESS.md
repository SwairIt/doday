## Принятые ключевые решения (с обоснованием)

| Решение | Дата | Почему |
|---|---|---|
| Отказ от AI-выполнения ДЗ | 2026-05-02 | Сломанная бизнес-модель, юридические риски, доступность бесплатных альтернатив |
| Веб-сайт как первая платформа | 2026-05-02 | Низкий порог входа, любое устройство |
| Монетизация через родительский кабинет | 2026-05-02 | Платежеспособный клиент — родитель, не школьник |
| Первый дневник — authedu.mosreg.ru | 2026-05-02 | Школа главного разработчика → он будет dogfood-пользователь |
| DiarySource абстракция с MVP | 2026-05-02 | Дешёвое добавление новых дневников потом vs дорогой рефакторинг |
| Python + FastAPI + HTMX | 2026-05-02 | Сила разработчика — Python; HTMX позволяет обойтись без React, что критично т.к. JS слабый |
| Российский хостинг | 2026-05-02 | 152-ФЗ требует локализации ПДн граждан РФ |
| Ручная вставка токена в MVP | 2026-05-02 | Просто и надёжно; авторасширение для браузера — после MVP |
| Родительский кабинет — после MVP | 2026-05-02 | Без живых юзеров некому тестировать; сначала проверим что школьники пользуются |
