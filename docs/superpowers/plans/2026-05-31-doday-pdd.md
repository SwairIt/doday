# Doday ПДД Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `getdoday.ru/pdd/` — free SEO-indexed official ПДД tickets (category АВМ) plus a Telegram-Stars Pro layer (mistake trainer, exam simulator, stats, weak-topics PDF), monetized via a standalone `pdd_pro` entitlement.

**Architecture:** A new per-feature module `app/pdd/` mirroring `app/qa/` (models / schemas / service / router / seo / seed_load), routers wired in `app/main.py`, templates under `app/templates/pdd/`. Monetization is added **additively** to `app/billing/` via a generic `Entitlement` table and a new `Product.grants_entitlement` field, so existing tier products and their ~850 tests are untouched. The bot payment handlers are already product-agnostic and need no changes.

**Tech Stack:** FastAPI, async SQLAlchemy 2.0, Alembic, Pydantic v2, Jinja2 + Tailwind CDN + Alpine + HTMX, fpdf2 (PDF), pytest. Spec: `docs/superpowers/specs/2026-05-31-doday-pdd-design.md`.

**Conventions (verified in-repo):**
- Routers go through `service`, never touch ORM directly (see `app/qa/router.py`).
- Tier gating: `app/billing/service.py` (`effective_tier`, `require_pro`, `has_pro_features`).
- Payment apply: `app/billing/stars.py::apply_successful_payment` (sets `user.tier`/`user.pro_until`).
- Tests build schema from `Base.metadata.create_all` in `tests/conftest.py` (each feature's `models` module imported there); between tests TRUNCATE. No migration needed for tests.

```python
class Entitlement(Base):
    """A per-user, per-feature access grant — generic so each new Doday vertical
    (pdd_pro, and future exam/interview verticals) reuses the same table with a
    different `feature` key. Independent of the global user.tier."""

    __tablename__ = "entitlements"

    id: Mapped[UUID] = mapped_column(primary_key=True, default=uuid4)
    user_id: Mapped[UUID] = mapped_column(
        ForeignKey("users.id", ondelete="CASCADE"), nullable=False, index=True
    )
    feature: Mapped[str] = mapped_column(String(40), nullable=False)
    # NULL = lifetime; otherwise the grant is active while expires_at is in the future.
    expires_at: Mapped[datetime | None] = mapped_column(DateTime(timezone=True), nullable=True)
    # Catalog code that granted/extended this entitlement (audit/forensics).
    source_code: Mapped[str] = mapped_column(String(50), nullable=False)
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), default=_utcnow, nullable=False
    )
    updated_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), default=_utcnow, onupdate=_utcnow, nullable=False
    )

    __table_args__ = (UniqueConstraint("user_id", "feature", name="uq_entitlement_user_feature"),)
```

async def has_entitlement(session: AsyncSession, user: User, feature: str) -> bool:
    """True if beta override is on, or the user holds a non-expired entitlement."""
    from app.billing.models import Entitlement
    from app.config import get_settings

    if get_settings().beta_free_for_all:
        return True
    row = await session.execute(
        select(Entitlement).where(Entitlement.user_id == user.id, Entitlement.feature == feature)
    )
    ent = row.scalar_one_or_none()
    if ent is None:
        return False
    return ent.expires_at is None or ent.expires_at > datetime.now(UTC)
```

async def _grant_entitlement(session: AsyncSession, user_id: UUID, product: Product) -> None:
    from app.billing.models import Entitlement

    ent = (
        await session.execute(
            select(Entitlement).where(
                Entitlement.user_id == user_id, Entitlement.feature == product.grants_entitlement
            )
        )
    ).scalar_one_or_none()
    if product.duration_months is None:
        new_expiry = None  # lifetime
    else:
        now = datetime.now(UTC)
        if ent is not None and ent.expires_at is not None and ent.expires_at > now:
            base = ent.expires_at
        else:
            base = now
        new_expiry = base + timedelta(days=30 * product.duration_months)
    if ent is None:
        session.add(
            Entitlement(
                user_id=user_id, feature=product.grants_entitlement,
                expires_at=new_expiry, source_code=product.code,
            )
        )
    else:
        # lifetime never downgrades to dated
        if ent.expires_at is None:
            return
        ent.expires_at = new_expiry
        ent.source_code = product.code
```
