# TG-каналы — pitch для разных групп

Telegram-каналы — твой главный канал в РФ. Подача разная для разных аудиторий: в Python-чате один tone, в индихакер-канале другой, у школьников третий.

**Принцип:** в чат пишешь **только когда тема релевантна**. Чистый «hello, у меня проект» = бан/игнор. Жди когда кто-то задаст вопрос про todo / FastAPI / HTMX / Mini App / индихакеры — тогда вступай с конкретным ответом и ссылкой на Doday.

Также: **в большие каналы лучше пиши админу**, не публикуй сам — у них есть правила анти-спам.

## 1) Python / FastAPI чаты

### Целевые каналы:

| Канал | Подписчики (примерно) | Тип |
|---|---|---|
| `@ru_python` | ~10k | Чат |
| `@pythonhub` | ~30k | Канал-новости |
| `@python_jobs_ru` | ~15k | Job-канал (не для нас) |
| `@fastapi_chat` (если есть) | ~5k | Чат |
| `@async_python` | ~5k | Чат |

### Подача — pitch для админа канала (newslettter / featured)

```
Здравствуйте.

Меня зовут Ярослав. Сделал todo-app Doday за 2 недели, мне 15 —
сейчас живёт на getdoday.ru, открытый код:
github.com/SwairIt/doday (MIT).

Стек который может быть интересен подписчикам:
FastAPI 0.115 + async SQLAlchemy 2.0 + Pydantic v2 + HTMX,
mypy --strict на 38 модулях, 650 тестов, ruff с E,F,I,UP,B,S,A,RUF.

Хабр-статья по проекту: habr.com/ru/articles/<id>

Если есть формат «проект недели» / featured-подборка — буду
благодарен если вставите.

Подходящие хэштеги или формат можно прислать — скажите.
```

### Подача — в открытом чате (когда уместно)

Сценарий: кто-то спрашивает «как лучше делать SPA на Python?» / «есть ли альтернатива React?» / «как сделать Telegram бот».

Твой ответ:

```
HTMX + FastAPI работает отлично для маленько-средних SPA. Я
2 недели назад запустил todo с тремя surface'ами (web + TG Mini
App + бот) — getdoday.ru, код github.com/SwairIt/doday. mypy
--strict проходит, deploy = git push. Если интересно глянуть
код или задать вопросы — спрашивай.
```

**Не больше 1 раза в неделю** в один чат с такой подачей. Иначе модератор посчитает спамом.

## 4) Школьные / молодёжные / олимпиадные чаты

### Где искать:

- **Чаты твоего класса/параллели** — самое близкое, начни оттуда
- VK-группы школы / района
- Олимпиадные чаты на codeforces / informatics.ru / Codewars-ru
- Программа «Сириус» / Yandex.Лицей / JetBrains Education — есть чаты выпускников
- TG-каналы для подростков-программистов (`@junior_devs_ru`, `@coding_jr_ru` если есть)

### Подача (специфика — твой ровесник, твой tone):

```
Я школьник 9 класса (15 лет). За свободное время в мае запустил
свой todo-app — getdoday.ru. Сделал его потому что Школьный
портал МО — кринж UX. У меня там домашка, проекты по школе,
свои задачи — всё в одном месте, plus Mini App в Telegram чтоб
открывать с тапа.

Бесплатно, можешь зарегаться и потыкать. Если найдёшь баг —
напиши мне, починю. Код открыт: github.com/SwairIt/doday.

Если кому-то полезно — буду рад. Если кому-то скучно — okey,
просто хочу делиться чем занимаюсь.
```

**Принцип в школьных чатах:** не как «продаю», а как «делюсь, мне интересно ваше мнение». Подростки чувствуют sales-tone мгновенно и закрываются.

## Чек-лист перед сообщением в любой чат

- [ ] Прочёл правила чата (есть ли rules для самопромоции?)
- [ ] Канал/чат активный (последний пост < 7 дней?)
- [ ] Уместная тема дня (если в чате обсуждают MAX-мессенджер — не лезь с todo, жди дискуссию про задачи)
- [ ] Сообщение **на «ты»** в чате / **на «Вы»** в DM админа
- [ ] Без spam-эмодзи в начале (🚀🚀🚀 = инст-игнор)
- [ ] Ссылка одна, не три (`getdoday.ru` ИЛИ `github.com/SwairIt/doday`, не оба сразу в одном сообщении)
- [ ] Подпись `— Ярослав, @SwairIt` чтобы было ясно с кем дело

## Что если забанили / удалили пост

- Не возвращайся в тот же чат с jail-account'a — точно навсегда. Подожди 3-6 месяцев, попробуй снова с другим углом.
- Учись из этого — что было не так? Тон? Тема? Время? Длина?
- Иди в следующий чат с скорректированной подачей.
