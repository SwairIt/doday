## Код — НЕ скриншотами

По просьбе автора весь код в статье — обычными код-блоками (` ``` `), которые
можно просто скопировать и вставить в редактор Хабра. Никаких carbon.now.sh,
сторонних сервисов и экспортов. Все нужные сниппеты (monkey-patch DNS,
`_HardcodedIPBackend`, `Product` + `apply_successful_payment`, структура папок,
таблица стека) уже лежат в тексте статьи готовыми блоками.
