# Как записать demo.gif для README

**Цель:** 30-секундный GIF (или MP4) показывающий ключевые фичи Mini App. Используется в README, Хабр-статье, постах в TG-канале.

## Что показать (сценарий — 30 секунд)

| t | Действие |
|---|---|
| 0:00 | Открываем `@DodayTaskBot` → тапаем menu-кнопку «Doday» (открывается Mini App) |
| 0:03 | Видим Today — несколько задач |
| 0:05 | Тапаем «+ Добавить задачу» → пишем «Купить молоко завтра !!!» → отправляем |
| 0:10 | Свайп задачи **влево** (Завтра — оранжевая) |
| 0:13 | Свайп задачи **вправо** (Готово — зелёная, конфетти) |
| 0:17 | Тап на задачу → открывается bottom-sheet с деталями |
| 0:19 | Скролл sheet вниз → Pomodoro/Подзадачи/Комментарии (раскрыть Комментарии, добавить «Готово ✓») |
| 0:25 | Закрываем sheet, идём в таб «Я» → видна статистика, переключатель темы |
| 0:28 | Тапаем «☀️ Светлая» → UI меняется → fin |

## Как записать

### Windows (рекомендую — ты на Windows)

**Вариант A: ScreenToGif** (бесплатный, прямо в GIF):
1. Download → https://www.screentogif.com/
2. Запусти Telegram Desktop в окне ~360×640 (mobile-aspect)
3. ScreenToGif → Recorder → выдели окно
4. FPS = 20, цвета = 128, размер GIF в итоге ~3-5 MB
5. Жми ⏺ → проигрываешь сценарий → ⏹
6. Editor → можешь обрезать лишние кадры → File → Save As → `demo.gif`

**Вариант B: OBS + ezgif.com** (если хочешь MP4 + GIF):
1. OBS Studio → Record окно Telegram Desktop
2. Сохрани MP4
3. ezgif.com → Video to GIF → Upload MP4 → 20 fps, ширина 480px → Convert
4. ezgif → Optimize → ≤5 MB

### Альтернатива — реальный телефон + AZ Screen Recorder

Если на mobile (свайпы натуральнее на тачскрине):
1. Android: AZ Screen Recorder; iPhone: встроенный Control Center recorder
2. Записать в MP4 → перекинуть на ПК → конвертировать в GIF на ezgif.com

## Финальные правки

- Размер файла: **≤ 5 MB** (GitHub разрешает рендерить inline, иначе тяжёлая загрузка README)
- Размер кадра: **480-540 px** широкая сторона (читаемо в README, не overweight)
- Длительность: **20-35 секунд** (короче — не успеть показать; длиннее — лента листает)
- Не нужно audio (GIF и так без звука)
