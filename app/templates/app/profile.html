Импорт домашки из браузера

Сервер не может достучаться до портала (видимо, VPN-расширение работает только в браузере). Поэтому ты копируешь JSON в браузере, а Doday уже создаёт задачи у себя.

  1. Открой портал в браузере, войди.
  2. F12 → таб Network → пройди в раздел «Домашние задания».
  3. Найди в списке запрос с именем типа homeworks / homework / tasks.
  4. Кликни → таб Response → правый клик → Copy value (или Ctrl+A → Ctrl+C).
  5. Вставь сюда ⬇ и жми «Импортировать».
