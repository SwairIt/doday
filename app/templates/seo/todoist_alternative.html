Частые вопросы

Как перенести задачи из Todoist?

Прямого импорта пока нет (на roadmap). Workaround: в Todoist → Настройки → Экспорт → CSV. Из CSV копируешь столбец с задачами → в Doday в Inbox → Quick-add → Bulk-paste (Ctrl+V). Распарсит до 200 строк за раз на Pro, 50 на Free. Даты и приоритеты тоже улавливает.

Доступен ли API для интеграций?

Да — REST API на FastAPI с автогенерируемой OpenAPI-схемой на /docs. Можешь писать свой скрипт-импорт, синхронизатор с другими сервисами, бот-помощник. Авторизация по cookie или token-based для CI.

Кто за этим стоит?

Один разработчик (мне 15). Doday — не корпоративный продукт, а инди. Это значит: быстрые фичи, ответ на письма от автора лично, и нет давления «inсreаse MRR на 20% за квартал» которое порождает дарк-паттерны.

Что если завтра ты закроешь сервис?

У всех есть JSON-бэкап аккаунта одним кликом (Настройки → Резервная копия). Плюс репозиторий open source — поднять Doday у себя на VPS можно за час (FastAPI + PostgreSQL, инструкция в README). Завязка на меня минимальная.
