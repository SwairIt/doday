Под какие предметы заточено

🇬🇧 Иностранные языки

Английский, немецкий, французский, испанский, китайский. Возможность задать разные цены для уровней (A1 / B2 / C1) — просто создайте несколько услуг.

📝 IELTS / TOEFL / экзамены

Пробное занятие 30 мин дешевле + основное 90 мин. Lead-time 24 часа — клиент готовит вопросы заранее. Пакет 8 занятий со скидкой.

🧮 Математика и физика

Длинные занятия (90–120 мин) для глубокого разбора задач. Описание услуги — какой класс / какие темы / какой формат материала.

🎓 Школа / ЕГЭ / ОГЭ

Подготовка к экзаменам — серии занятий по чёткому графику. CSV-выгрузка для отчётов родителям. Daily digest помогает не пропустить ни одного ученика.
