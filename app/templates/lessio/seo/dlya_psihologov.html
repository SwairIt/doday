Под какие практики

🧠 Психотерапия / личная терапия

Услуги по 50 мин, lead-time 24+ часа, буфер 15-30 мин между клиентами на запись заметок. Карточка клиента хранит только историю записей, не содержимое.

💭 Коучинг

Длительные сессии (60-90 мин), пакеты 4/8/12 сессий. Подходит для life-coaching, executive-coaching, спортивного коучинга.

👥 Парная и семейная терапия

Используйте групповую услугу с max_attendees=2 чтобы клиенту нужно было записать обоих участников отдельно. Или просто примите по одному.

🌱 Супервизии

Скрытая услуга (не появится в списке для всех), ссылку даёте только верифицированным коллегам. Пакеты со скидкой через цены.
